How It Works

1. Cookie Consent Banner

When users first visit, they'll see a cookie consent banner at the bottom of the page. They can either:

  • Accept All - Enables all cookies including advertising
  • Manage Preferences - Choose which cookies to enable

2. Google AdSense Ads

Google ads (with class .ad-slot-google) will only load if the user accepts advertising cookies. This ensures GDPR compliance.

3. Partner Ads

Electa partner ads (with class .ad-slot-partner) are always visible and don't require consent since they're not personalized or tracked.

4. Testing

To test the cookie consent flow:

  1. Open this page in a private/incognito window
  2. You should see the cookie banner
  3. Try both "Accept All" and "Manage Preferences"
  4. Check that Google ads appear only after accepting advertising cookies

Code Examples

Example 1: Standard Responsive Ad

<div class="ad-slot-google ad-slot-top">
  <ins class="adsbygoogle"
       style="display:block"
       data-ad-client="ca-pub-1234567890123456"
       data-ad-slot="9876543210"
       data-ad-format="auto"
       data-full-width-responsive="true"></ins>
</div>
